## Installing the OwnID Passwordless plugin
To download and install the OwnID plugin, access the official [plugin page](https://wordpress.org/plugins/ownid-passwordless-login/), or search for it directly in the `WordPress > Plugins` admin menu of your WordPress instance. 

## Configuration Steps
An OwnID application connects your WordPress website with the OwnID button in the front end. To create an OwnID application:
1. Open the [OwnID Console](https://console.ownid.com/) and create an account or log in to an existing account.
2. Create a **WordPress or WooCommerce** integration and complete the onboarding.
3. Copy and paste the **appID** and **shared secret** into the settings of the OwnID WordPress plugin.

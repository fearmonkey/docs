## Enrollment Endpoint

In your frontend code, after a user logged in successfully, call to the above endpoint in order to fetch the `authToken`.

Make a request to the backend endpoint and as a response they will pass the `authToken` that they will utilize in the following promise.

```Javascript
await ownid.enrollCredential(loginId, authToken);
```

## Install the OwnID Plugin for WordPress
To download and install the OwnID plugin, access the official WordPress [plugin page](https://wordpress.org/plugins/ownid-passwordless-login/), or search for it directly in the `WordPress > Plugins` admin menu of your WordPress instance.
